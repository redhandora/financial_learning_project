# 📚 金融知识系统学习计划

## 🎯 项目概述

这是一个完整的金融知识学习系统，专为每天有1小时学习时间的初学者设计。计划从最基础的经济学概念开始，逐步深入到专业金融知识，为期12个月。

## 📚 学习计划三个阶段

### 第一阶段：基础入门（第1-3个月）
- **目标**：建立金融思维框架
- **内容**：经济学基础、会计基础、货币银行学
- **每日**：1小时学习 + 思考练习

### 第二阶段：核心课程（第4-7个月）
- **目标**：掌握金融核心理论
- **内容**：公司理财、投资学
- **每日**：1小时深入学习 + 案例分析

### 第三阶段：专业化方向（第8-12个月）
- **目标**：选择专业方向深入
- **方向**：投资分析、量化金融、风险管理、国际金融
- **每日**：1小时专业学习 + 项目实践
