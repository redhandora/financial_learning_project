📖 核心概念

1. 需求（Demand）

定义：在一定时期内，消费者愿意并且能够购买的商品或服务的数量。

需求定律

在其他条件不变的情况下，商品价格上升，需求量减少；价格下降，需求量增加。

需求曲线

向右下方倾斜的曲线，表示价格与需求量之间的反向关系。

需求曲线示意图
价格
↑
│   ↘
│     ↘
│       ↘ 需求曲线
│         ↘
│           ↘
└─────────────→ 数量

2. 供给（Supply）

定义：在一定时期内，生产者愿意并且能够提供的商品或服务的数量。

供给定律

在其他条件不变的情况下，商品价格上升，供给量增加；价格下降，供给量减少。

供给曲线

向右上方倾斜的曲线，表示价格与供给量之间的正向关系。

供给曲线示意图
价格
↑
│           ↗
│         ↗
│       ↗ 供给曲线
│     ↗
│   ↗
└─────────────→ 数量

3. 市场均衡（Market Equilibrium）

定义：需求量和供给量相等时的状态。

均衡价格

使需求量等于供给量的价格。

均衡数量

均衡价格下的交易量。

市场调节机制

  • 当价格高于均衡价格：供给量 > 需求量 → 价格下降
  • 当价格低于均衡价格：需求量 > 供给量 → 价格上升
市场均衡示意图
价格
↑
│         ↗ 供给曲线
│       ↗
│     ↗   × 均衡点
│   ↗   ↙
│ ↗   ↙
│   ↙     ↘ 需求曲线
│ ↙       ↘
└─────────────→ 数量

📅 每日学习安排

周一：供给与需求概念

基础概念

学习内容：

  • 阅读《经济学原理》第4章：供给与需求的市场力量
  • 理解需求表和需求曲线
  • 理解供给表和供给曲线

关键要点：

  1. 需求曲线的移动 vs 需求量的变化
  2. 供给曲线的移动 vs 供给量的变化
  3. 影响需求和供给的因素

思考题：

  • 为什么需求曲线向右下方倾斜？
  • 为什么供给曲线向右上方倾斜？
  • 哪些因素会导致需求曲线移动？

周二：需求弹性

计算分析

学习内容：

  • 需求价格弹性的概念
  • 弹性计算公式
  • 弹性的五种类型

计算公式：

需求价格弹性 = (需求量变化百分比) / (价格变化百分比)

弹性类型：

完全无弹性 弹性 = 0
缺乏弹性 0 < 弹性 < 1
单位弹性 弹性 = 1
富有弹性 弹性 > 1
完全弹性 弹性 = ∞

思考题：

  • 生活必需品和奢侈品的弹性有什么不同？
  • 为什么时间越长，需求弹性越大？
  • 如何利用弹性制定定价策略？

周三：市场均衡

案例分析

学习内容：

  • 均衡价格和数量的确定
  • 市场失衡的调整过程
  • 消费者剩余和生产者剩余

案例分析：

假设某商品的市场情况：

  • 需求函数：Qd = 100 - 2P
  • 供给函数：Qs = 20 + 3P
计算任务：
  1. 均衡价格和数量
  2. 如果政府设定最高价格P=15，会发生什么？
  3. 如果政府设定最低价格P=25，会发生什么？

思考题：

  • 为什么市场价格会趋向均衡？
  • 政府价格管制会带来什么后果？
  • 黑市为什么会产生？

周四：消费者剩余

概念应用

学习内容：

  • 消费者剩余的概念
  • 如何计算消费者剩余
  • 消费者剩余与市场效率

消费者剩余：

消费者愿意支付的最高价格与实际支付价格之间的差额。

计算公式：

消费者剩余 = 需求曲线以下、价格线以上的面积

思考题：

  • 消费者剩余如何衡量消费者的福利？
  • 价格下降如何影响消费者剩余？
  • 完全竞争市场为什么能最大化消费者剩余？

周五：综合练习

综合应用

练习题：

1. 画出以下情况的供给需求图：
  • 技术进步降低了生产成本
  • 消费者收入增加
  • 替代品价格上升
  • 互补品价格下降
2. 计算分析：

某商品需求函数：Qd = 80 - 4P

供给函数：Qs = -20 + 6P

  1. 求均衡价格和数量
  2. 如果政府征收每单位2元的税，新的均衡是什么？
  3. 税收负担如何在消费者和生产者之间分配？

思考题：

  • 总结本周学习的关键概念
  • 供给需求理论在现实生活中的应用
  • 你观察到哪些市场现象可以用供给需求解释？

💡 促进思考的活动

1. 现实观察练习

任务：观察你日常生活中的一个市场（如超市、菜市场、网购平台）

记录要点：

  1. 商品价格的变化规律
  2. 不同时间段的需求变化
  3. 促销活动对销售的影响
  4. 季节性因素对供给的影响

分析要求：

  • 用供给需求理论解释观察到的现象
  • 预测价格未来可能的变化
  • 思考商家如何利用供给需求规律

2. 案例分析：口罩市场（2020年）

背景：2020年新冠疫情爆发初期，口罩需求急剧增加。

分析问题：

  1. 需求曲线如何移动？
  2. 供给曲线如何移动？
  3. 均衡价格和数量如何变化？
  4. 政府价格管制的影响？
  5. 长期供给如何调整？

思考方向：

  • 市场如何应对突发需求冲击？
  • 价格机制在资源配置中的作用？
  • 政府干预的利弊？

3. 思维训练：价格信号

游戏规则：

  1. 假设你是苹果种植者
  2. 去年苹果价格很高
  3. 今年你应该种植更多还是更少苹果？
  4. 其他种植者会怎么想？
  5. 最终市场会如何调整？

思考问题：

  • 价格如何传递信息？
  • 个体决策如何影响整体市场？
  • "看不见的手"如何工作？
