## 📖 核心概念

### 1. 市场均衡的数学表达
**需求函数**：Qd = a - bP （a>0, b>0）
**供给函数**：Qs = c + dP （c可正可负，d>0）

**均衡条件**：Qd = Qs
**均衡价格**：P* = (a - c) / (b + d)
**均衡数量**：Q* = a - bP* = c + dP*

### 2. 市场失衡与调整
**超额需求**：Qd > Qs → 价格上升压力
**超额供给**：Qs > Qd → 价格下降压力

**调整过程**：
1. 价格信号传递信息
2. 买卖双方调整行为
3. 市场趋向新的均衡
4. 资源重新配置

### 3. 供需变化对均衡的影响

| 变化类型 | 需求增加 | 需求减少 | 供给增加 | 供给减少 |
|---------|----------|----------|----------|----------|
| 均衡价格 | 上升 | 下降 | 下降 | 上升 |
| 均衡数量 | 增加 | 减少 | 增加 | 减少 |

**同时变化分析**：
- 供需同向变化：数量确定，价格不确定
- 供需反向变化：价格确定，数量不确定

### 4. 政府干预类型
1. **价格上限**（Price Ceiling）：最高限价
   - 目的：保护消费者
   - 效果：可能导致短缺、黑市、质量下降
   
2. **价格下限**（Price Floor）：最低限价
   - 目的：保护生产者
   - 效果：可能导致过剩、政府收购、资源浪费
   
3. **数量管制**：配额、许可证
4. **税收与补贴**：改变相对价格

### 5. 消费者剩余与生产者剩余
**消费者剩余**（CS）：需求曲线以下、价格线以上的面积
**生产者剩余**（PS）：供给曲线以上、价格线以下的面积
**总剩余**（TS）：CS + PS，衡量市场效率

**市场效率**：完全竞争市场使总剩余最大化

## 📅 每日学习安排

### 周一：均衡的数学分析
**学习内容**：
- 需求函数和供给函数的数学表达
- 均衡价格和数量的计算
- 供需变化对均衡的影响计算

**计算练习**：
市场情况：
- 需求：Qd = 100 - 2P
- 供给：Qs = 20 + 3P

计算：
1. 均衡价格和数量
2. 如果需求增加到 Qd = 120 - 2P，新均衡？
3. 如果供给减少到 Qs = 10 + 3P，新均衡？
4. 如果同时发生上述变化，新均衡？

**思考题**：
- 为什么用线性函数简化分析？
- 现实中的需求供给函数是线性的吗？
- 如何估计真实市场的函数参数？

### 周二：市场调整机制
**学习内容**：
- 价格的信息传递功能
- 市场失衡的自动调整
- 调整速度的影响因素
- 市场摩擦的影响

**案例分析**：房地产市场调整
1. 需求突然增加（如政策刺激）
2. 供给短期固定（建设需要时间）
3. 价格快速上涨
4. 长期供给增加
5. 价格逐步稳定

**思考题**：
- 为什么有些市场调整快，有些调整慢？
- 信息不完全如何影响调整？
- 预期在调整中起什么作用？

1. 计算市场均衡
