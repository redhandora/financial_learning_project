## 📖 核心概念

### 1. 消费者剩余（Consumer Surplus, CS）
**定义**：消费者愿意支付的最高价格与实际支付价格之间的差额。

**经济意义**：
- 衡量消费者从交易中获得的净收益
- 反映消费者的福利水平
- 用于评估市场效率和政策效果

**计算方法**：
- 线性需求：CS = ½ × (最高支付意愿 - 市场价格) × 数量
- 一般情况：需求曲线以下、价格线以上的面积

### 2. 生产者剩余（Producer Surplus, PS）
**定义**：生产者实际得到的价格与愿意接受的最低价格之间的差额。

**经济意义**：
- 衡量生产者从交易中获得的净收益
- 反映生产者的福利水平
- 激励生产和创新

**计算方法**：
- 线性供给：PS = ½ × (市场价格 - 最低接受价格) × 数量
- 一般情况：供给曲线以上、价格线以下的面积

### 3. 总剩余（Total Surplus, TS）
**定义**：CS + PS，衡量社会总福利。

**计算公式**：
```
TS = CS + PS = (消费者总支付意愿 - 实际支付总额) + (实际收入总额 - 生产者总成本)
```

**市场效率**：完全竞争市场使TS最大化

### 4. 市场效率条件
**帕累托效率**：无法使任何人境况变好而不使他人境况变坏的状态。

**完全竞争市场的效率特征**：
1. 资源配置有效：价格 = 边际成本
2. 生产有效：在最低成本下生产
3. 交换有效：商品分配给评价最高的消费者

### 5. 市场失灵类型
1. **市场势力**：垄断、寡头
2. **外部性**：污染、技术溢出
3. **公共物品**：国防、路灯
4. **信息不对称**：二手车市场、保险
5. **不完全竞争**：进入壁垒、产品差异

## 📅 每日学习安排

### 周一：消费者剩余深入
**学习内容**：

- 支付意愿与实际支付
- 消费者剩余的计算方法

**计算练习**：
需求函数：Qd = 100 - 2P
1. 当P=20时，计算CS
2. 当P从20降到15时，CS变化多少？

**思考题**：
