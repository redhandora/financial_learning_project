# 📚 第1周：供给与需求基础

## 🎯 本周学习目标
1. 理解供给与需求的基本概念
2. 掌握市场均衡的形成机制
3. 能够分析价格如何调节市场
4. 应用供给需求理论分析现实问题

## 📖 核心概念

### 1. 需求（Demand）
**定义**：在一定时期内，消费者愿意并且能够购买的商品或服务的数量。

**需求定律**：在其他条件不变的情况下，商品价格上升，需求量减少；价格下降，需求量增加。

**需求曲线**：向右下方倾斜的曲线，表示价格与需求量之间的反向关系。

### 2. 供给（Supply）
**定义**：在一定时期内，生产者愿意并且能够提供的商品或服务的数量。

**供给定律**：在其他条件不变的情况下，商品价格上升，供给量增加；价格下降，供给量减少。

**供给曲线**：向右上方倾斜的曲线，表示价格与供给量之间的正向关系。

### 3. 市场均衡（Market Equilibrium）
**定义**：需求量和供给量相等时的状态。

**均衡价格**：使需求量等于供给量的价格。
**均衡数量**：均衡价格下的交易量。

**市场调节机制**：
- 当价格高于均衡价格：供给量 > 需求量 → 价格下降
- 当价格低于均衡价格：需求量 > 供给量 → 价格上升

### 周二：需求弹性
**学习内容**：
- 需求价格弹性的概念
- 弹性计算公式
- 弹性的五种类型

**计算公式**：
```
需求价格弹性 = (需求量变化百分比) / (价格变化百分比)
```

**弹性类型**：
1. 完全无弹性（弹性=0）
2. 缺乏弹性（0<弹性<1）
3. 单位弹性（弹性=1）
4. 富有弹性（弹性>1）
5. 完全弹性（弹性=∞）

**思考题**：
- 生活必需品和奢侈品的弹性有什么不同？
- 为什么时间越长，需求弹性越大？
- 如何利用弹性制定定价策略？

### 周三：市场均衡
**学习内容**：
- 均衡价格和数量的确定
- 市场失衡的调整过程
- 消费者剩余和生产者剩余

**案例分析**：
假设某商品的市场情况：
- 需求函数：Qd = 100 - 2P
- 供给函数：Qs = 20 + 3P

计算：
1. 均衡价格和数量
2. 如果政府设定最高价格P=15，会发生什么？
3. 如果政府设定最低价格P=25，会发生什么？

**思考题**：
- 为什么市场价格会趋向均衡？
- 政府价格管制会带来什么后果？
- 黑市为什么会产生？

### 周四：消费者剩余
**学习内容**：
- 消费者剩余的概念
- 如何计算消费者剩余
- 消费者剩余与市场效率

**消费者剩余**：消费者愿意支付的最高价格与实际支付价格之间的差额。

**计算公式**：
```
消费者剩余 = 需求曲线以下、价格线以上的面积
```

**思考题**：
- 消费者剩余如何衡量消费者的福利？
- 价格下降如何影响消费者剩余？
- 完全竞争市场为什么能最大化消费者剩余？

## 💡 促进思考的活动

### 1. 现实观察练习
**任务**：观察你日常生活中的一个市场（如超市、菜市场、网购平台）

**记录**：
1. 商品价格的变化规律
2. 不同时间段的需求变化
3. 促销活动对销售的影响
4. 季节性因素对供给的影响

**分析**：
- 用供给需求理论解释观察到的现象
- 预测价格未来可能的变化
- 思考商家如何利用供给需求规律

### 2. 案例分析：口罩市场（2020年）
**背景**：2020年新冠疫情爆发初期，口罩需求急剧增加。

**分析问题**：
1. 需求曲线如何移动？
2. 供给曲线如何移动？
3. 均衡价格和数量如何变化？
4. 政府价格管制的影响？
5. 长期供给如何调整？

**思考**：
- 市场如何应对突发需求冲击？
- 价格机制在资源配置中的作用？
- 政府干预的利弊？

### 3. 思维训练：价格信号
**游戏规则**：
1. 假设你是苹果种植者
2. 去年苹果价格很高
3. 今年你应该种植更多还是更少苹果？
4. 其他种植者会怎么想？
5. 最终市场会如何调整？

**思考**：
- 价格如何传递信息？
- 个体决策如何影响整体市场？
- "看不见的手"如何工作？
