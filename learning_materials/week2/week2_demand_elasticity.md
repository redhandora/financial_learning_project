2. 掌握影响需求弹性的因素

4. 应用弹性理论制定商业策略
5. 理解弹性与税收负担的关系

## 📖 核心概念

### 1. 需求价格弹性（Price Elasticity of Demand）
**定义**：衡量需求量对价格变化的敏感程度。

**计算公式**：
```
Ed = (ΔQ/Q) / (ΔP/P) = (百分比需求量变化) / (百分比价格变化)
```

**中点公式**（更准确）：
```
Ed = [(Q2 - Q1) / ((Q1 + Q2)/2)] / [(P2 - P1) / ((P1 + P2)/2)]
```

### 2. 弹性类型与特征

| 弹性类型 | 弹性值 | 特征 | 例子 |
|---------|--------|------|------|
| 完全无弹性 | Ed = 0 | 需求量不随价格变化 | 救命药、必需品 |
| 缺乏弹性 | 0 < Ed < 1 | 需求量变化比例小于价格变化比例 | 食品、汽油 |
| 单位弹性 | Ed = 1 | 需求量变化比例等于价格变化比例 | 理论情况 |
| 富有弹性 | Ed > 1 | 需求量变化比例大于价格变化比例 | 奢侈品、旅游 |
| 完全弹性 | Ed = ∞ | 价格微小变化导致需求量无限变化 | 完全竞争市场 |

### 3. 影响需求价格弹性的因素
1. **替代品的可获得性**：替代品越多，弹性越大
2. **必需品 vs 奢侈品**：必需品弹性小，奢侈品弹性大
3. **占收入比例**：占收入比例大，弹性大
4. **时间长短**：时间越长，弹性越大
5. **商品定义范围**：定义越窄，弹性越大

### 4. 收入弹性（Income Elasticity of Demand）
**定义**：衡量需求量对收入变化的敏感程度。

**计算公式**：
```
Ei = (ΔQ/Q) / (ΔI/I)
```

**类型**：
- 正常商品：Ei > 0
  - 必需品：0 < Ei < 1
  - 奢侈品：Ei > 1
- 劣等商品：Ei < 0

### 5. 交叉价格弹性（Cross-Price Elasticity）
**定义**：衡量一种商品需求量对另一种商品价格变化的敏感程度。

**计算公式**：
```
Exy = (ΔQx/Qx) / (ΔPy/Py)
```

**类型**：
- 替代品：Exy > 0
- 互补品：Exy < 0
- 无关商品：Exy = 0

## 📅 每日学习安排

### 周一：需求价格弹性基础
**学习内容**：
- 弹性的概念和意义
- 弹性计算公式
- 弹性的五种类型

**计算练习**：
1. 价格从10元降到8元，需求量从100增加到120，计算弹性
2. 价格从5元涨到6元，需求量从200降到180，计算弹性
3. 使用中点公式重新计算上述两种情况

**思考题**：
- 为什么弹性用百分比而不是绝对值？
- 中点公式为什么更准确？
- 弹性值的经济意义是什么？

### 周二：影响弹性的因素
**学习内容**：
- 替代品可获得性
- 商品性质（必需品/奢侈品）
- 占收入比例
- 时间因素

**案例分析**：
分析以下商品的弹性：
1. 食盐
2. 汽油
3. 高端智能手机
4. 出国旅游
5. 特定品牌的咖啡

**思考题**：
- 为什么汽油短期弹性小，长期弹性大？
- 如何判断一个商品是必需品还是奢侈品？
- 技术进步如何影响商品弹性？

### 周三：收入弹性和交叉弹性
**学习内容**：
- 收入弹性的概念和计算
- 正常商品 vs 劣等商品
- 交叉价格弹性的概念和计算
- 替代品 vs 互补品

**计算练习**：
1. 收入增加20%，某商品需求量增加10%，计算收入弹性
2. 商品B价格上升10%，商品A需求量增加15%，计算交叉弹性
3. 判断商品关系：替代品、互补品还是无关

**思考题**：
- 收入增加时，哪些商品需求会增加？哪些会减少？
- 如何利用交叉弹性进行产品定位？
- 吉芬商品是什么？为什么存在？

### 周四：弹性与总收益
**学习内容**：
- 弹性与总收益的关系
- 价格变化对总收益的影响
- 最优定价策略

**重要关系**：
- 缺乏弹性（Ed < 1）：价格上升 → 总收益增加
- 富有弹性（Ed > 1）：价格上升 → 总收益减少
- 单位弹性（Ed = 1）：价格变化不影响总收益

**案例分析**：
某商品当前价格P=100，需求量Q=1000，弹性Ed=0.8
1. 如果价格提高到110，总收益如何变化？
2. 如果价格降低到90，总收益如何变化？
3. 最优价格应该是多少？

**思考题**：
- 为什么电影院对学生打折？
- 为什么航空公司对商务舱定价高？
- 如何利用弹性制定促销策略？

### 周五：弹性与税收负担
**学习内容**：
- 税收对市场的影响
- 税收负担的分配
- 弹性与税收归宿

**税收负担分配**：
税收负担在消费者和生产者之间的分配取决于需求弹性和供给弹性：
- 需求越缺乏弹性，消费者负担比例越大
- 供给越缺乏弹性，生产者负担比例越大

**计算练习**：
某商品市场：
- 需求：Qd = 100 - 2P
- 供给：Qs = 20 + 3P
政府征收每单位2元的税，计算：
1. 新的均衡价格和数量
2. 消费者支付的价格
3. 生产者得到的价格
4. 税收负担分配

**思考题**：
- 为什么香烟税主要由消费者承担？
- 为什么劳动税可能由雇主和雇员共同承担？
- 如何设计税收政策减少扭曲？

**游戏**：弹性侦探

**规则**：
1. 观察一个价格变化事件（如涨价、降价、促销）
2. 预测需求量的变化
3. 估算弹性值
4. 验证预测结果

**案例**：
- 咖啡店涨价10%
- 健身房推出年卡优惠
- 电子产品新款发布，旧款降价
- 节假日旅游产品价格调整

**思考**：
- 如何更准确预测弹性？
- 哪些信息对预测弹性最重要？
- 弹性预测的局限性？

**背景**：某国考虑对奢侈品征收高额税收

**分析问题**：
1. 奢侈品的需求弹性特征
2. 税收对消费者行为的影响

4. 税收收入预测
5. 可能的 unintended consequences

**思考**：
- 奢侈品税能达到预期目标吗？
- 谁最终承担税负？
- 有没有更好的替代政策？
