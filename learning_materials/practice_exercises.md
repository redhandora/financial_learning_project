# 📝 金融学习练习题与思考题集

## 第1阶段：基础入门（第1-4周）

### 第1周：供给与需求基础

#### 基础练习题
1. **需求曲线绘制**
   - 已知需求表：P=10时Q=100，P=9时Q=110，P=8时Q=120
   - 画出需求曲线
   - 计算价格从10降到8的需求量变化百分比

2. **供给曲线绘制**
   - 已知供给表：P=5时Q=50，P=6时Q=60，P=7时Q=70
   - 画出供给曲线
   - 计算价格从5升到7的供给量变化百分比

3. **市场均衡计算**
   - 需求函数：Qd = 80 - 4P
   - 供给函数：Qs = -20 + 6P
   - 求均衡价格和数量

#### 进阶思考题
1. **现实观察分析**
   - 观察超市中某种商品的价格变化
   - 记录一周内价格和销量的变化
   - 用供给需求理论解释观察到的现象
   - 预测未来价格可能的变化

2. **政策影响分析**
   - 假设政府对汽油征收每升1元的税
   - 分析短期和长期对市场的影响
   - 讨论税收负担的分配
   - 提出替代政策建议

3. **弹性思维训练**
   - 列出5种商品，估计它们的需求弹性
   - 解释估计依据
   - 设计实验验证你的估计
   - 讨论估计误差的原因

### 第2周：需求弹性深入

#### 基础练习题
1. **弹性计算**
   - 价格从100元降到90元，需求量从1000增加到1200
   - 计算需求价格弹性
   - 判断弹性类型
   - 分析价格变化对总收益的影响

2. **中点公式练习**
   - 使用中点公式计算以下情况：
     - P1=20, Q1=100; P2=18, Q2=120
     - P1=50, Q1=200; P2=55, Q2=180
   - 比较中点公式和普通公式的结果差异

3. **交叉弹性分析**
   - 商品A和商品B是替代品
   - 商品B价格上升10%，商品A需求量增加15%
   - 计算交叉价格弹性
   - 分析商业策略启示

#### 进阶思考题
1. **定价策略设计**
   - 选择一种你熟悉的商品或服务
   - 分析其需求弹性特征
   - 设计三种不同的定价策略
   - 预测每种策略的效果
   - 提出实施建议

2. **税收政策评估**
   - 研究现实中的一种税收（如烟草税、碳税）
   - 分析税收的弹性基础
   - 评估税收效果（收入、行为改变、福利影响）
   - 提出改进建议

3. **弹性调研设计**
   - 设计一个简单的需求弹性调研
   - 确定调研对象和方法
   - 设计调研问卷
   - 规划数据分析方法
   - 讨论调研的局限性和改进

### 第3周：市场均衡深入分析

#### 基础练习题
1. **供需变化分析**
   - 初始市场：Qd=100-2P, Qs=20+3P
   - 分析以下变化对均衡的影响：
     a) 需求增加到Qd=120-2P
     b) 供给减少到Qs=10+3P
     c) 同时发生a和b

2. **价格管制计算**
   - 市场：Qd=80-2P, Qs=-10+3P
   - 计算市场均衡
   - 政府设定价格上限P=20
   - 计算短缺数量
   - 分析可能后果

3. **福利分析基础**
   - 市场：Qd=100-P, Qs=20+2P
   - 计算市场均衡的CS、PS、TS
   - 价格上限P=25时的CS、PS、TS
   - 计算效率损失

#### 进阶思考题
1. **现实政策分析**
   - 选择一个现实的价格管制政策（如租金管制、最低工资）
   - 收集相关数据和信息
   - 分析政策效果
   - 评估政策效率
   - 提出政策改进建议

2. **市场模拟设计**
   - 设计一个简单的市场模拟游戏
   - 确定参与角色和规则
   - 设计不同政策场景
   - 规划数据收集和分析
   - 讨论模拟的教育价值

3. **效率思维训练**
   - 观察一个存在效率损失的市场现象
   - 分析效率损失的原因
   - 估算效率损失的大小
   - 设计效率改进方案
   - 讨论改进方案的可行性

### 第4周：消费者剩余与市场效率

#### 基础练习题
1. **消费者剩余计算**
   - 需求函数：Qd = 60 - 3P
   - 市场价格P=10
   - 计算消费者剩余
   - 如果价格降到P=8，CS变化多少？

2. **生产者剩余计算**
   - 供给函数：Qs = -20 + 4P
   - 市场价格P=15
   - 计算生产者剩余
   - 如果价格升到P=18，PS变化多少？

3. **税收福利分析**
   - 市场：Qd=80-2P, Qs=20+3P
   - 政府对每单位征收t=5的税
   - 计算税后均衡
   - 计算税收负担分配
   - 计算无谓损失

#### 进阶思考题
1. **政策福利评估**
   - 选择一个现实的补贴政策（如新能源汽车补贴、农业补贴）
   - 分析政策目标和机制
   - 评估福利效果（CS、PS、政府成本）
   - 计算政策效率
   - 提出评估结论和建议

2. **市场失灵分析**
   - 识别一个市场失灵案例
   - 分析失灵的原因和表现
   - 评估福利损失
   - 设计政府干预方案
   - 讨论干预的效果和成本

3. **效率公平权衡**
   - 选择一个涉及效率公平权衡的政策问题
   - 分析效率目标和公平目标
   - 设计权衡方案
   - 评估方案效果

## 综合案例分析题

### 案例1：共享单车市场分析

**背景**：
2016-2018年，中国共享单车市场经历爆发式增长。ofo和摩拜等公司大量投放单车，价格战激烈，随后出现倒闭潮。

**数据**：
- 初始需求：Qd = 1000 - 50P（P为每次骑行价格，单位：元）
- 初始供给：Qs = 200 + 30P
- 资本涌入后供给：Qs = 500 + 30P
- 价格战期间价格：P = 0.5元/次
- 正常运营成本：约1元/次

**问题**：
1. 计算初始市场均衡
2. 分析资本涌入对市场的影响
3. 评估价格战期间的市场状况
4. 分析企业倒闭的经济原因
5. 提出行业健康发展建议

### 案例2：房地产市场调控

**背景**：
某城市为控制房价上涨，出台限购、限贷、限价政策。

**数据**：
- 需求函数：Qd = 8000 - 20P（P为房价，单位：万元/平米）
- 供给函数：Qs = 2000 + 10P
- 政策前均衡：P=200, Q=4000
- 限价政策：P≤180
- 限购政策：需求减少20%

**问题**：
1. 分析限价政策的市场影响
2. 分析限购政策的市场影响
3. 评估政策组合效果
4. 分析可能出现的 unintended consequences
5. 设计更有效的调控方案

### 案例3：碳税政策设计

**背景**：
为应对气候变化，政府考虑征收碳税。

**数据**：
- 高碳行业需求：Qd = 100 - P
- 高碳行业供给：Qs = 20 + 2P（未考虑环境成本）
- 碳排放成本：每单位产品排放2吨CO₂
- 碳税税率考虑：100元/吨CO₂、200元/吨CO₂、300元/吨CO₂
- 低碳替代品价格：60元/单位

**问题**：
1. 分析不同税率对市场的影响
2. 计算各税率的福利效果
3. 分析产业转移风险
4. 设计补偿机制
5. 提出政策实施建议

## 计算题答案模板

### 题目1：市场均衡计算
**题目**：Qd = 80 - 4P, Qs = -20 + 6P

**解答**：
1. 均衡条件：Qd = Qs
2. 80 - 4P = -20 + 6P
3. 80 + 20 = 6P + 4P
4. 100 = 10P
5. P* = 10
6. Q* = 80 - 4×10 = 40
   （或 Q* = -20 + 6×10 = 40）

**答案**：均衡价格P*=10，均衡数量Q*=40

### 题目2：弹性计算
**题目**：价格从100降到90，需求量从1000增加到1200

**解答**：
1. 价格变化百分比：ΔP/P = (90-100)/100 = -10%
2. 需求量变化百分比：ΔQ/Q = (1200-1000)/1000 = +20%
3. 需求价格弹性：Ed = 20% / 10% = 2.0
4. 弹性类型：富有弹性（Ed>1）
5. 总收益变化：原收益=100×1000=100,000
   新收益=90×1200=108,000
   增加8,000

**答案**：Ed=2.0，富有弹性，总收益增加

### 题目3：消费者剩余计算
**题目**：Qd = 60 - 3P，P=10

**解答**：
1. 当P=10时，Q=60-3×10=30
2. 最高支付意愿：令Q=0，0=60-3P，P=20
3. 消费者剩余：CS = ½ × (20-10) × 30 = 150

**答案**：消费者剩余CS=150

## 学习建议

### 练习题使用指南
1. **循序渐进**：从基础题开始，逐步挑战进阶题
2. **定时练习**：每周安排固定时间做练习
3. **错题整理**：建立错题本，定期复习
4. **举一反三**：做完题后思考变式问题
5. **实际应用**：将练习题与现实问题结合

### 思考题使用指南
1. **深度思考**：不满足于表面答案，深入分析
2. **多角度分析**：从不同角度思考问题
3. **证据支持**：用数据和事实支持观点
4. **批判性思维**：质疑假设，评估论证
5. **创造性解决**：提出创新性解决方案

### 案例题使用指南
1. **系统分析**：使用系统框架分析案例
2. **数据驱动**：基于数据做出判断
3. **现实联系**：将案例分析与现实联系
4. **方案设计**：设计具体可行的方案
5. **效果评估**：评估方案的可能效果

### 思维训练指南
1. **日常训练**：将思维训练融入日常生活
2. **记录思考**：记录思考过程和结论
3. **反思改进**：定期反思思考方法
4. **交流讨论**：与他人讨论，拓宽思路
5. **持续学习**：不断学习新的思考工具
