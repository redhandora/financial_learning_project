# 🎯 金融知识系统学习计划（每日1小时）

## 📊 计划概览
**总时长**：12个月（约360小时）
**每日学习**：1小时（工作日）+ 2小时（周末一天）
**学习节奏**：循序渐进，从基础到专业

## 📚 三个阶段详细安排

### 第一阶段：基础入门（第1-3个月）
**目标**：建立金融思维框架，理解基本概念

**月度安排**：
- 第1个月：经济学基础（《经济学原理》曼昆）
- 第2个月：会计与金融概论
- 第3个月：货币银行学（《货币金融学》米什金）

**每日1小时结构**：
```
前10分钟：回顾前一天内容
中间40分钟：学习新内容
后10分钟：整理笔记+思考题
```

### 第二阶段：核心课程（第4-7个月）
**目标**：掌握金融核心理论，能够进行基本分析

**月度安排**：
- 第4-5个月：公司理财（《公司理财》罗斯）
- 第6-7个月：投资学（《投资学》博迪）

**每日1小时结构**：
```
前5分钟：快速回顾
中间45分钟：深入学习+案例分析
后10分钟：应用思考+记录问题
```

### 第三阶段：专业化方向（第8-12个月）
**目标**：选择专业方向深入，形成个人分析框架

**可选方向**：
1. 投资分析与价值投资
2. 量化金融与金融工程
3. 风险管理
4. 国际金融

**每日1小时结构**：
```
前10分钟：阅读专业新闻/研究报告
中间40分钟：专业技能学习/项目实践
后10分钟：专业思考/职业规划
```

## 💡 促进思考的学习活动

### 1. 每周案例分析（30分钟）
使用哈佛分析框架分析真实金融事件

### 2. 每日思考题系统
- 基础阶段：为什么货币有时间价值？
- 核心阶段：如何用NPV评估投资项目？
- 专业阶段：如何用DCF模型估值公司？

### 3. 思维训练游戏
- 财富公式建模（理解复利）
- 风险骰盅（平衡风险收益）
- 市场侦探（培养分析思维）

### 4. 实践项目
- 虚拟投资组合（持续12个月）
- 个人财务分析（第2个月开始）
- 行业研究报告（第6个月开始）

### 5. 七把财商思维钥匙
每周训练一把：交换思维、稀缺思维、刚需思维、远见思维、跃迁思维、协作思维、概念思维

## 🎯 学习建议

### 成功关键：
1. **坚持每日学习**：养成习惯比学习时长更重要
2. **主动思考**：多问"为什么"，理解背后逻辑
3. **理论联系实际**：每个概念都要思考现实应用
4. **建立知识体系**：融会贯通，形成框架

### 常见误区：
- ❌ 过早追求复杂理论
- ❌ 只看不练，缺乏实践
- ❌ 忽视数学和统计基础
- ❌ 只关注短期市场波动

### 阶段性里程碑：
- **3个月后**：能看懂财经新闻，理解基本术语
- **7个月后**：能进行基本投资分析，看懂财务报表
- **12个月后**：形成个人投资理念或分析框架

### 第1周学习计划：

| 星期 | 学习内容 | 具体安排 | 思考题 |

| 周五 | 综合练习 | 完成章节练习题 | 总结本周关键概念 |
